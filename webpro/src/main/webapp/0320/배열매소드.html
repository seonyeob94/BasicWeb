indexOf(item, start) item의 위치찾기
lastIndexOf(item) 역순으로 item의 위치찾기



push(a,b,c...) : 배열의 마지막에 요소를 추가
pop() : 배열의 마지막 요소를 제거하고 반환한다



arrNum = [10,23,100,115,200,5]
sort() : 요소를 사전순으로 배열
숫자일 경우는 크기순으로 변경 가능하다



slice(start,end) : start~end-1까지의 요소를 추출
새로운 배열로 생성
원래 배열은 변하지 않는다



splice(index, n, a,b,c...): index 위치에서 n개를 삭제하고 a,b,c요소를 삽입한다
n의 값이 0이면 삭제되는 요소는 없다
a,b,c요소를 생략하면 삽입되는 요소가 없다



a.concat(b,c) : a 배열에 b,c 배열을 추가해서 하나의 배열로 만든다



join : 배열을 하나의 문자열로 변환하고 리턴한다
